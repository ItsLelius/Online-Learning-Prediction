# Online-Learning-Prediction
A data mining and information management project that predicts student course completion behavior in an online learning platform using classification, clustering, and association rule mining.

## About
This project simulates a real enterprise data workflow combining 
Information Management (IM) and Data Mining (DM) to predict 
whether a student will complete, drop, or remain ongoing in an 
online course.

## Tech Stack
- Python (Pandas, Scikit-learn, Matplotlib, Seaborn)
- MySQL (Database Design and Schema)
- Google Colab (Notebook Environment)
- GitHub (Version Control)

## Techniques Used
- Data Cleaning and Preprocessing
- Principal Component Analysis (PCA)
- Classification (Decision Tree, kNN, Naive Bayes, Logistic Regression, Random Forest)
- Clustering (K-Means)
- Association Rule Mining (Apriori)
